# Eridian_companion_bot_Eleni._.
This robot has 5 legs and at least 8 degrees of freedom, it has a OLED screen that can show cute faces, temp and custom information. Has a remote control mode that is linked to a controller but also could just be a desk top buddy with automatic responses to temp change, tap on the head and time.
