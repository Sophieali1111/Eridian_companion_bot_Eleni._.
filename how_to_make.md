# electronic components 

all brough online and shipped in can refrence the BOM including : 
- ESP32 development board
- LiPo battery 3.7V 1500mAh
- TP4056 charging module
- Capacitive touch sensor
- 9g micro servos ×10
- OLED SSD1306
- AAA batteries
- Toggle switch
- DHT22 sensor
- 10k resistors
- Battery box 4×AA
- 5V boost converter
- Jumper wires (owned)
